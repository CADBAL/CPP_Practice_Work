# What’s the difference between a variable’s scope, duration, and lifetime? By default, what kind of scope and duration do local variables have (and what do those mean)?

# A variable's scope determines where that variable can be accessed in the code. A variable's duration are the rules that determine when it is created and destroyed. A variable's lifetime is the time between when it was created and when it was destroyed.
# This distinction is important because a variable could be created at the very end of a block but it's scope does not care when it was created in that block, it just cares that the scope was that block.

# Local variables scope is within the blocks they are initialized, it has block scope. 
# Local variables have automatic duration, which means they are created at the point of definition, and destroyed at the end of the block in which they are defined.
